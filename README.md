# Unit-Converter

The unit converter is a solo project from Scrimba javascript course.

The unit converter only converts the below metrics:
- Length
    - Meters to Feets
    - Feets to Meters
- Volumes
    - Liters to Gallons
    - Gallons to Liters
- Mass
    - Kilograms to Pounds
    - Pounds to Kilograms
